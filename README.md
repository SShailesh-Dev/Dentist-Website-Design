# 🦷 Dentist Website Design

A modern, responsive website project built with **HTML, CSS, TypeScript, and Vite**.  
This project demonstrates clean UI design principles, accessibility, and performance optimization tailored for a dental clinic.

---

## ✨ Features
- **Responsive Layout**: Works seamlessly across desktop, tablet, and mobile devices.  
- **Modern Tooling**: Powered by Vite for fast development and optimized builds.  
- **Scalable Structure**: Organized `src/` folder with reusable components and clear guidelines.  
- **Professional Aesthetic**: Clean typography, calming color palette, and user‑friendly navigation.  

---

## 🎯 Purpose
This project was created as part of my portfolio to showcase my ability to design and develop professional websites for healthcare businesses.  
It highlights my skills in front‑end development, project structuring, and modern web workflows.
